# Shck

tool for ssh and terminal multiplexier

## Installation

Add this line to your application's Gemfile:

    gem 'shck'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shck

## Usage

    $ vi ~/.shck.yml
    $ shck
